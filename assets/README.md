# Small repository assets

This directory is reserved for small, redistribution-safe figures or diagrams
used by the documentation.

Do not place checkpoints, datasets, third-party weights, generated rollout
collections, or metric caches here. Public model assets will live in the
[CD-LAM Hugging Face repository](https://huggingface.co/yufanwei/CD-LAM) after
upload and verification. SAM3 and CoWTracker assets remain under their own
licenses and storage conventions.
